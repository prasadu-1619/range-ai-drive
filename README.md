# Range AI Drive

A sophisticated Electric Vehicle (EV) simulation platform that helps users understand and optimize their EV's range performance using AI-driven analytics.

## Features

- 3D visualization of EV performance
- Real-time range prediction
- Interactive controls and HUD
- Terrain impact analysis
- AI-powered range optimization

## Tech Stack

- React + TypeScript
- Three.js for 3D visualization
- Supabase for backend
- Tailwind CSS + Shadcn UI
- Vite for build tooling

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

```
src/
├── components/     # React components
├── hooks/         # Custom React hooks
├── integrations/  # Third-party integrations
├── lib/          # Utility functions
└── pages/        # Application pages
```
